# Usage

```html
<i class="u-icon-save"/>
```

In case use icon attribute in `UButton` and `UIcon` class `u` will automatically added
```vue
<template>
  <div>
    <u-button icon="u-icon-save"/>
    <u-icon icon="u-icon-save"/>
  </div>
</template>
```
