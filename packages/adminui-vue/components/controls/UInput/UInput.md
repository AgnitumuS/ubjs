### Basic usage

samples below are binded to `req_Request` entity

```vue
<template>
  <u-input attribute-name="notes" />
</template>
```

### When attribute type is number

```vue
<template>
  <u-input attribute-name="cost" />
</template>
```

### Disabled

```vue
<template>
  <u-input
    attribute-name="notes"
    disabled
  />
</template>
```
